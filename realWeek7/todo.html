    Why is the JavaScript textContent property safer than innerHTML?

    textContent is safer than innerHTML because it does not parse HTML tags.

    Explain the relationship between an HTML attribute and a corresponding DOM property (explain the connection to JavaScript). Use a specific example in your answer

    HTML attributes are the properties of HTML elements. For example, the href attribute of an <a> tag is a property of the DOM element. In JavaScript, you can access this property using document.getElementById("myLink").href.
